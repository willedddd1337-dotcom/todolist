# 📝 Todo List App

Простий веб-додаток для управління списком завдань, побудований на Flask.

## 🚀 Функціонал

- Перегляд поточного списку завдань
- Додавання нових завдань через форму
- Видалення завдань за індексом

## 🛠️ Технології

- **Python 3**
- **Flask** — легкий веб-фреймворк
- **Jinja2** — шаблонізатор (вбудований у Flask)

## 📦 Встановлення

1. Клонуй репозиторій:
   ```bash
   git clone https://github.com/your-username/todo-list.git
   cd todo-list
   ```

2. Створи та активуй віртуальне середовище:
   ```bash
   python -m venv venv
   source venv/bin/activate      # Linux/macOS
   venv\Scripts\activate         # Windows
   ```

3. Встанови залежності:
   ```bash
   pip install flask
   ```

## ▶️ Запуск

```bash
python app.py
```

Відкрий браузер і перейди за адресою: [http://localhost:5000](http://localhost:5000)

## 📁 Структура проєкту

```
todo-list/
├── app.py

    └── index.html
```

## 🔌 API / Маршрути

| Метод | Маршрут            | Опис                          |
|-------|--------------------|-------------------------------|
| GET   | `/`                | Головна сторінка зі списком   |
| POST  | `/add`             | Додати нове завдання          |
| GET   | `/delete/<int:id>` | Видалити завдання за індексом |

## ⚠️ Примітки

> Дані зберігаються **лише в пам'яті** (список `todos`). Після перезапуску сервера всі завдання скидаються до початкових значень. Для постійного зберігання можна додати базу даних (наприклад, SQLite через Flask-SQLAlchemy).
